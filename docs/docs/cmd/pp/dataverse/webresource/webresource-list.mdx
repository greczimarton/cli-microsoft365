# pp dataverse webresource list

Lists web resources in the specified Power Platform solution

## Usage

```sh
m365 pp dataverse webresource list [options]
```

## Options

```md definition-list
`-e, --environmentName <environmentName>`
: The Id of the Power Platform environment
`-s, --solutionId <solutionId>`
: The Id of the Power Platform solution

`--excludeContent`
: Exclude the base64 content of the web resources
```

<Global />

## Examples

List the web resources in a specified Microsoft Power Platform solution

```sh
pp dataverse webresource list --environmentName '4be50206-9576-4237-8b17-38d8aadfaa41' --solutionId '19b25bde-9eb7-4cb2-914d-cd8a4717ee84'
```

List the web resources in a specified Microsoft Power Platform solution without the base64 content of each web resource

```sh
pp dataverse webresource list --environmentName '4be50206-9576-4237-8b17-38d8aadfaa41' --solutionId '19b25bde-9eb7-4cb2-914d-cd8a4717ee84' --excludeContent
```
